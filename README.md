# Description

**`isa-concepts`** is an ISA Simulator — Python implementation of a custom 16-bit Half Precision ISA with binary conversion, memory/register storage, addressing modes, an instruction compiler, and a program runner.

```
Created as a laboratory project for CMSC 132 Computer Architecture.
```
